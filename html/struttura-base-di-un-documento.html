Struttura base di un documento HTML

Ogni pagina web ha bisogno di una struttura chiara e corretta per essere interpretata dai browser e dai motori di ricerca. Vediamo come costruirla nel modo giusto.

Indice

  • 1. La dichiarazione DOCTYPE
  • 2. L’elemento <html lang="it">
  • 3. Il blocco <head>
  • 4. Il blocco <body>
  • 5. Esempio completo

1. La dichiarazione DOCTYPE

Ogni documento HTML deve iniziare con la dichiarazione del tipo di documento. In HTML5 è:

<!DOCTYPE html>

Questa linea indica al browser che il documento usa lo standard HTML5.

2. L’elemento <html lang="it">

L’elemento <html> racchiude tutto il contenuto HTML della pagina. L’attributo lang="it" specifica che il contenuto è in italiano, utile per l’accessibilità e per i motori di ricerca.

3. Il blocco <head>

Contiene le informazioni sulla pagina (metadati) non visibili nel corpo del documento. Include:

  • <meta charset="UTF-8">: definisce la codifica dei caratteri
  • <title>: il titolo della pagina, visibile nella scheda del browser
  • <meta name="description">: una descrizione per la SEO
  • <meta name="viewport">: necessaria per il layout responsive
  • <link rel="stylesheet">: per collegare il foglio di stile

4. Il blocco <body>

Contiene tutti gli elementi visibili della pagina: testi, immagini, pulsanti, form, ecc. È il cuore del contenuto HTML.

5. Esempio completo

<!DOCTYPE html>
<html lang="it">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Esempio HTML base</title>
  </head>
  <body>
    <h1>Benvenuto in HTML!</h1>
    <p>Questa è una pagina di esempio.</p>
  </body>
</html>

Questa è la base da cui partire per qualsiasi progetto web.
